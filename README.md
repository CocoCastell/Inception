# Description
This project is an introduction to system administration using the tool Docker.
In particular, Inception is a project focused on deploying a complete web stack using Docker containers and Docker compose.
The goal is to create a fully functional WordPress site running with Nginx in HTTPS, connected to a MariaDB database.
